DOM is Document Object Model. It is a programming interface of documents in web which is used as nodes and object to interact with all pages. We can also update the structure, style or content of a document.

How will you select HTML elements using DOM? Name the DOM methods? There are 5 way in which we can select elements in DOM. They are: getElementsByTagName() : This method is about to select elements by their tag name. getElementsByClassName() : This method returns a collection of elements if it has specified class. getElementById() : This allow to select an element by a unique id to returns the element with the specified id. querySelector() : When a CSS selector is on and matched with elements it gives the first element only. querySelectorAll() : If a CSS selector is on and matched with elements it gives back the collection of elements.

Event bubbling is a function in DOM or Document Object Model that occurs an event in a particular element in the HTML document.

A Callback Function is a function that executed after a function has finished executing. And we use it because when a function needs to wait for a process to complete it before continuing another execution.
